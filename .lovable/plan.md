
# 🎰 数字抽取转盘游戏

## 页面布局

### 顶部区域
- 应用标题，带有渐变色彩和动画效果
- 副标题说明游戏规则

### 中间核心区域 - 转盘展示
- **圆形转盘**：展示1-15的数字，采用炫彩渐变配色
- **中央抽取按钮**：点击触发随机分配动画
- 转盘旋转动画效果，增加趣味性

### 玩家区域
三个并排的玩家卡片区域，每个包含：
- **角色头像/图标**：可爱的角色图标
- **角色名称**：可编辑的输入框，用户可自定义名称
- **数字卡片网格**：展示分配到的5个数字
  - 每张卡片采用鲜艳的渐变色背景
  - 卡片带有悬停动画效果
  - 每张卡片右上角有消除按钮（X图标）

### 底部操作区
- **重新抽取按钮**：重置所有数字，重新分配
- **重置按钮**：清空所有结果，恢复初始状态

## 交互功能

1. **抽取动画**：点击抽取时，转盘旋转 + 数字卡片逐个飞入各玩家区域
2. **消除功能**：点击卡片上的X按钮，卡片以淡出动画消失
3. **角色编辑**：点击角色名称可直接编辑
4. **悬停效果**：卡片悬停时放大并显示阴影

## 视觉设计风格

- 🌈 **配色方案**：采用彩虹渐变色系（紫色、蓝色、青色、绿色、黄色、橙色、红色）
- ✨ **背景**：浅色渐变背景带有微妙的几何图案
- 🎴 **卡片设计**：圆角卡片，带有玻璃态毛玻璃效果
- 💫 **动画**：流畅的过渡动画和微交互
- 📱 **响应式**：适配不同屏幕尺寸
